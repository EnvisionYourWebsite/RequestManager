How to install

Please before installing the code

  • Create a empty database
  • If You want to use Facebook Login:
    • Create an Application on Facebook Developer Web site https://developers.facebook.com/apps
    • Copy your App ID, App Secret
  • Navigate to http:://yourdomainame/install 
  • Follow steps 
 

How to upgrade

Upgrade from 1.0

  • Navigate to http://yourdomainname/install/index.php/upgrader/
  • If You want to use Facebook Login:
    • Create an Application on Facebook Developer Web site https://developers.facebook.com/apps
    • Copy your App ID, App Secret
  • Paste your AppID, App Secret in the inputs
  • Submit

Upgrade to 1.3

In this version you need to convert the database and tables to UTF-8 (utf8_unicode_ci) (Unicode Support)

Upgrade to 1.4 & 2.0

Just Extract it to your server

Upgrade to 2.5 & 2.6 & 2.6.1

  • Extract the archive into your server
  • Navigate to http://yourdomainname/install/index.php/upgrader/
  • Done

How to use

Users can suggest ideas & feedbacks or vote to existing ideas,

Admins can apply a status for an idea to inform users (only the interested users) that idea is accepted and to keep them updated about the development of the new feature.

 

Suggest a new idea

  • Enter your idea to the text input
  • Describe it 
  • (2.5) you can put them in a category
  • Enter your email & password 
  • Submit 

Vote on exiting idea 

  • Simply click on vote button 

 

Admin can apply a status to an idea 

  • Select the idea that you want to apply a status
  • set status  (users will be notified by mail about the changes)

PS: Categories are created by admin in their dashboard
